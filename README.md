# Practico Evaluativo -- PP1-Python-Javascript

En este repositorio se encontraran las carpetas frontend (Javascript) y backend (Python).

## Pasos a seguir backend

* Clonar el repositorio.

```bash
git clone git@github.com/pe-pp1-python-javascript.git
```

* Ir a la carpeta backend.

```bash
cd backend_py
```

* Crear entorno virtual (venv).

```bash
python3 -m venv venv
```

* Activar el entorno e instalar dependencias de FastAPI.

```bash
source venv/bin/activate
pip install -r requirements.txt
```

* Trabajando en el proyecto.

```bash
fastapi dev main.py
```
